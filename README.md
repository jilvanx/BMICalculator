## :hammer_and_wrench: Techs
* Kotlin
* Activity: To screen host
* ViewGroup
  * RelativeLayout
  * LinearLayout
- Components de UI
  - TextView
  - EditText
  - Button
- Toast: To show messages
- Intent: Explain how to open a second Activity
- FindViewById: To retrieve UI components
- String resource: For concatenation of texts
